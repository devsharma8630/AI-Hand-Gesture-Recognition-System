# AI Hand Gesture Recognition System

This project is a real-time AI-based hand gesture shape recognition system that detects and classifies multiple hand gestures using webcam.

## Features
- Detects 10+ hand gesture shapes
- Real-time webcam recognition
- Uses MediaPipe hand landmarks
- Built with Python, OpenCV & AI
